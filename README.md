# Knowledge Garden

Knowledge Garden is an academic collaboration and resource platform designed to support students, faculty, and educational institutions in creating, discovering, sharing, and engaging with scholarly content. It combines intelligent resource management, real-time collaboration tools, personalized search, and insightful analytics within a secure, scalable, and user-friendly environment.
With features like auto-tagging, discussion forums, document annotations, and faculty verification, Knowledge Garden fosters a trusted, interactive learning ecosystem that adapts to individual learning journeys and institutional needs.

## Setup Instructions

This repository does not contain the source code itself but rather links to the actual frontend and backend repositories using submodules. To download the source code, first clone the repository and then inside the directory, run `git submodule init && git submodule update`, then follow the setup instructions of backend and frontend repositories.
